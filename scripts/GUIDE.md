# Скрипты тестирования воронки

Тестовый пользователь: `379336096`

## Запуск

Из корневой папки проекта:

```bash
./scripts/run.sh stage_0.py
```

Или напрямую из папки `bot/`:

```bash
cd bot && uv run python ../scripts/stage_0.py
```

Каждый скрипт:
1. Устанавливает `funnel_stage` в базе
2. Отправляет сообщение в Telegram точно как бот (с фото, кнопками, кружками)

## Callback-кнопки

Кнопки в сообщениях — настоящие inline-кнопки. Чтобы callback-кнопки работали при нажатии, бот должен быть запущен (`cd bot && uv run python -m src.main`).

- `video_workout` (stage 0) — отправляет ссылку на видео тренировки
- `learn_workout` (stage 1) — отправляет описание + кнопку оплаты 690₽
- `video_circle` (stage 3) — отправляет видео-кружок
- `buy_now` (stages 5, 6) — отправляет сообщение с кнопкой оплаты
